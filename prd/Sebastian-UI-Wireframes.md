# Sebastian UI Wireframes

**버전**: 2.0.0
**작성일**: 2025-12-12
**대상**: Claude Code (PyQt6 구현용)

---

## 디자인 원칙

**Extreme Minimalism**: 기능에 집중하는 전문가용 도구

- 불필요한 장식 없음
- 명확한 시각적 계층
- 충분한 여백
- 일관된 색상 체계
- 빠른 작업 완료가 목표

## PyQt6 구현 가이드

### 위젯 매핑

| UI 요소 | PyQt6 위젯 |
|---------|-----------|
| 메인 창 | `QMainWindow` |
| 탭 | `QTabWidget` |
| 버튼 | `QPushButton` |
| 입력 필드 | `QLineEdit` |
| 폴더 선택 | `QFileDialog.getExistingDirectory()` |
| 파일 선택 | `QFileDialog.getOpenFileNames()` |
| 프로그레스 | `QProgressBar` |
| 로그 | `QPlainTextEdit` |
| Dialog | `QDialog` |
| 체크박스 | `QCheckBox` |
| 라디오 버튼 | `QRadioButton` |

### 스타일시트 예시

```python
# 전역 스타일
app.setStyleSheet("""
    QWidget {
        font-family: "Pretendard", "맑은 고딕", sans-serif;
        font-size: 13px;
        color: #212121;
    }

    QPushButton {
        border-radius: 8px;
        padding: 8px 16px;
        background: #F5F5F5;
        border: 1px solid #E0E0E0;
    }

    QPushButton:hover {
        background: #EEEEEE;
    }

    QLineEdit {
        border: 2px solid #E0E0E0;
        border-radius: 4px;
        padding: 8px 12px;
        background: #FFFFFF;
    }

    QLineEdit:focus {
        border-color: #00897B;
    }
""")
```
